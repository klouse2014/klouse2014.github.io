使用GitHub＋Pelican搭建个人博客

github准备¶

  • 创建username.github.io 数据仓库

pelican准备¶

  • 新建blog文件件, 执行pelican-quickstart命令创建基本框架
  • 将output文件夹绑定到数据仓库
cd output
git init
git remote add origin https://github.com/username/username.github.io.git
git pull origin master
  • pelican 包含很多主题和插件，可以直接clone使用
    • pelican-themes
    • pelican-plugins

  • 使用elegant主题

    • elegant主题说明文档

  • 生成网页文件以及在本机上测试

make html
make serve

可以使用浏览器打开:http://localhost:8000来访问这个测试服务器, 然后就可以欣赏到你的博客了
